# Airwallex Take-Home Assignment

Built using React, ViteJS, TailwindCSS, DaisyUI, Vitest, React Testing Library, and MSW.

## Deployment

[https://sc-airwallex-takehome.netlify.app/](https://sc-airwallex-takehome.netlify.app/)

## Run

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Test

```sh
npm run test
```

## Coverage

```sh
npm run coverage
```
